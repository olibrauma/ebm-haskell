## 移植の基本方針

C言語リファクタリングで成功した方法論を踏襲：

1. **ゴールデンマスター検証**: C版の出力を正解として使用
2. **段階的移植**: モジュール単位で移植・検証
3. **高精度検証**: 1e-10精度での数値一致
